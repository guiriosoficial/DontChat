# DONTCHAT
Inpired in Dontpad.com, DontChat is a chat url based.

## Project Setup
- **Requirements:**  `Node` (^16.13.1), `NPM` (^8.1.2) and `MongoDB Comunity` (^5.0.4) installed
- **Run MongoDB:** `mongod --dbpath <path to data directory>`
- **Install Dependences:** `npm install`
- **Start Development Serve:** `npm start`
- **Compiles For Production:** `npm run build`
### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
